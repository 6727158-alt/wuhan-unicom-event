# H5活动小程序开发提示词（优化版）

## 角色定位
你是一名资深前端开发工程师，专精于移动端H5小程序开发。你需要使用HTML5 + CSS3 + JavaScript（或Vue3/React）开发高质量的活动预约小程序。

## 项目背景
- **客户**: 武汉联通
- **活动名称**: 2026年武汉联通合作伙伴大会
- **目标**: 开发手机端活动预约H5小程序
- **设计风格**: 红色主色调 + 科技感/商务风格 + iOS风格
- **技术栈**: 单页面应用(SPA)，响应式设计，适配移动端

---

## 页面结构（共7页）

### 页面1 - 封面页（Landing Page）
**功能**: 活动入口展示

**布局**:
- 顶部: 中国联通LOGO + 活动主题大字（红色）
- 中部: 活动信息（两行小字）
  - "活动时间：2026年4月20日"
  - "活动地点：武汉会议中心黄鹤厅"
- 底部: 红色主按钮"点击参加活动"

**交互**:
- 点击按钮 → 跳转页面2（首次用户）或页面3（已预约用户）
- 使用localStorage判断用户是否已预约

---

### 页面2 - 活动预约登记页
**功能**: 收集参会者信息

**布局**:
- 顶部标题区: LOGO + "2026武汉联通合作伙伴大会" + "活动预约登记"
- 表单区: 5个卡片式输入框
  1. 姓名（文本）
  2. 联系方式（手机号）
  3. 代表单位（文本）
  4. 所在行业（文本）
  5. 同行参会人数（数字）
- 底部: 红色提交按钮

**交互**:
- 表单验证: 所有字段必填，手机号格式校验
- 验证失败: 未填字段显示红色提示"*请填写预约信息"
- 验证成功: 
  - 显示成功弹窗（横向长方形、圆角、红色边框、半透明红底）
  - 弹窗文案: "预约信息提交成功"
  - 2秒后自动跳转页面3
  - 保存预约状态到localStorage

---

### 页面3 - 活动内容详情页（导航中心）
**功能**: 功能导航主页

**布局**:
- 顶部: LOGO + "2026武汉联通合作伙伴大会"
- 中部: 4个卡片按钮（田字格2×2排列）
  - 活动场地与流程（建筑图标）
  - 活动座位分区（座位图标）
  - 合作项目跟进（文件夹图标）
  - 活动评价（星星图标）

**样式**:
- 白色圆角正方形卡片
- 悬浮效果（hover/active状态）
- 图标与功能名匹配

**交互**:
- 点击卡片 → 跳转对应页面
- 所有子页面可返回此页

---

### 页面4 - 活动场地与流程
**功能**: 场地信息 + 活动日程

**布局**:
- 顶部: 返回按钮 + 页面标题
- 场地信息区:
  - 场地名称: "武汉会议中心黄鹤厅"
  - 3个信息卡片按钮:
    1. 参会路线图 → 弹窗显示图片
    2. 会场布局图 → 弹窗显示图片
    3. 会场详细地址 → 弹窗显示地址信息
- 活动流程区:
  - 标题: "活动流程"
  - 时间线形式展示（从上到下）:
    - 9:00 - 入场签到 📝
    - 9:25 - 活动开场 🎬
    - 9:30 - 领导致辞 🎤
    - 9:45 - 嘉宾发言 💼
    - 10:00 - 品牌推介 📊
    - 10:30 - 签约仪式 ✍️
    - 10:45 - 合影环节 📸
    - 11:00 - 活动结束 🏁

**弹窗**:
- 右上角有关闭按钮
- 路线图/布局图弹窗: 预留图片位置
- 地址弹窗: 显示完整地址和联系方式

---

### 页面5 - 活动座位分区
**功能**: 座位分布 + 展位信息

**布局**:
- 顶部: 返回按钮 + "活动座位分区"
- 座位分布图: 可视化座位区域（A/B/C/D区）
- 展位列表: 卡片式展示

**展位数据**（随机生成8个）:
- 展位号: A01, A02, A03...A08
- 公司名称: 华为、中兴、腾讯、阿里等
- 合作项目: 5G智慧园区、企业专线等
- 联系人 + 电话

---

### 页面6 - 合作项目跟进
**功能**: 展示合作项目和进度

**布局**:
- 顶部: 返回按钮 + 页面标题
- 项目卡片列表（随机生成5个）:
  - 项目名称
  - 所属公司
  - 对接负责人
  - 联系电话
  - 创建时间
  - 目前进展描述
  - 进度条（百分比）
- 底部: "新增项目跟进"按钮

**新增项目弹窗**:
- 表单字段: 项目名称、公司、负责人、电话、进展描述、进度百分比
- 确认添加按钮

---

### 页面7 - 活动评价
**功能**: 活动反馈收集

**布局**:
- 顶部: 返回按钮 + 页面标题
- 感谢语: "感谢您参加2026武汉联通合作伙伴大会"
- 评价表单:
  1. 整体满意度: 5星评分
  2. 评价内容: 多行文本框（100字限制）+ 字数统计
  3. 单项评分（5星制）:
     - 活动组织
     - 活动服务
     - 会场氛围
     - 活动内容
- 底部: 提交按钮

## 技术规范

### 文件结构
```
project/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── app.js          # 业务逻辑
└── images/             # 图片资源
```

### 样式要求
- 红色主色调: #e60012（中国联通红）
- 科技渐变: 红→紫→蓝渐变背景
- 卡片圆角: 16px
- 按钮圆角: 12px
- 阴影: 柔和投影增强层次感
- 动画: 页面切换过渡、按钮点击反馈、悬浮效果

### 交互规范
- 页面切换: 滑动动画效果
- 按钮状态: normal / hover / active
- 表单验证: 实时反馈，错误提示
- 弹窗: 遮罩层 + 圆角卡片 + 关闭按钮
- 返回逻辑: 子页面统一返回页面3

### 数据存储
- 使用localStorage存储:
  - 用户预约状态
  - 预约表单数据
  - 新增的项目数据
  - 评价数据
